# Linked Lancer

## Steps to Set Up 
- Clone
> ```bash
> git clone https://github.com/ihadabs/linked-lancer.git
> ```
- Go to Project Directory
> ```bash
> cd linked-lancer
> ``` 
- Install Packages
> ```bash
> npm i
> ```
- Run
> ```bash
> npm run dev
> ```
- See It on Your Browser
> ```
> localhost:3000
> ```
